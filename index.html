🔥 تعلم Godot (شرح حقيقي)

🎮 صنع اللاعب خطوة خطوة

🔹 الخطوة 1:

افتح Godot → New Scene → اختر CharacterBody2D

🔹 الخطوة 2:

أضف نودات:

  • Sprite2D
  • CollisionShape2D

🔹 الخطوة 3:

احفظ المشهد باسم Player.tscn

🔹 الخطوة 4 (الكود):

extends CharacterBody2D

@export var speed = 250

func _physics_process(delta):
    var dir = Input.get_axis("ui_left","ui_right")
    velocity.x = dir * speed
    move_and_slide()

🔹 الخطوة 5:

افتح Main Scene واسحب Player.tscn داخلها

🧟 صنع زومبي يلاحق اللاعب

🔹 الخطوة 1:

New Scene → CharacterBody2D

🔹 الخطوة 2:

أضف:

  • Sprite2D
  • CollisionShape2D

🔹 الخطوة 3:

احفظه Zombie.tscn

🔹 الخطوة 4 (الكود الصحيح):

extends CharacterBody2D

@export var speed = 100
var player

func _ready():
    player = get_parent().get_node("Player")

func _physics_process(delta):
    if player:
        var dir = (player.global_position - global_position).normalized()
        velocity = dir * speed
        move_and_slide()

🔹 الخطوة 5 (مهم جداً):

روح للمشهد الرئيسي Main واسحب:

  • Player.tscn
  • Zombie.tscn

📌 تأكد اسم اللاعب داخل المشهد هو: Player
